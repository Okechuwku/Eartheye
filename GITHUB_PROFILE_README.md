## 🧑‍💻 About Me

I’m **Okechuwku Emeka Joel**, a hands-on cybersecurity practitioner focused on **web security, API testing, reconnaissance, and bug bounty methodology**.

I enjoy mapping attack surfaces, testing real applications, and validating findings carefully before I call anything a vulnerability. My style is practical: **recon → identify → test → confirm → document**.

- 🔥 Focused on real-world security testing, not just theory
- ⚡ Building consistent workflows for manual and automated recon
- 🧠 Learning how attackers think so I can test like one, ethically
- 🛠️ Creating a portfolio of notes, scripts, and security tooling
