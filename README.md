# Eartheye

**Eartheye** is an AI-powered attack surface discovery and vulnerability scanning platform designed for penetration testers and security researchers. See your website through the eyes of artificial intelligence.

## Features
- **Attack Surface Mapping**
- **Automated Reconnaissance**
- **Vulnerability Detection**
- **Professional Pentesting**
- **Automated Bug Bounty Recon Engine**

## Tech Stack
- **Frontend:** React, TailwindCSS, Framer Motion
- **Backend:** Python FastAPI
- **Database:** PostgreSQL
- **Scanners:** `subfinder`, `httpx`, `katana`, `LinkFinder`, `ffuf`, `nuclei`

## Project Structure
- `/frontend` - React application
- `/backend` - FastAPI backend
- `/scanner` - Scanner execution pipeline
- `/database` - Database migrations & schemas
- `/scans` - Raw output from scanner tools
